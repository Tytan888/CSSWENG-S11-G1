adminController

This module contains all of the functions that are used to handle requests to the admin pages.

Properties:
Name Type Description
initializeAdmin function

Initializes the admin account if it does not exist.
adminLogin function

Renders the admin login page.
adminSubmit function

Checks if the admin login credentials are correct.
adminAuth function

Checks if the admin is logged using the user's cookies in and redirects to the 404 page if not.
adminMenu function

Renders the admin menu page.
adminMain function

Renders admin pages based on the type of page requested.
